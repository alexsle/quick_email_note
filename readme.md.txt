Quickly email a note to yourself from Android. Flow for Llamalab Automate App.

Install the app https://play.google.com/store/apps/details?id=com.llamalab.automate&referrer=utm_source%3Dhomepage

Use your email credentials.

Checks the phone is connected to Wi-Fi or Mobile network(s).
Simple message input. 
Checks network connection again.
Sends email with message as subject.
Shows confirmation msg with 1 sec timeout.

Create widget on your Android Homescreen, choose Automate flow
